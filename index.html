Smart Light

Smart Lighting System

Smart Light

Smart Lighting System

The objective of this project is to develop a smart lighting system that includes a user interface designed with contemporary web technologies. The system will be controlled by microcontroller technology, aiming to provide efficient management of the lighting environment.

The core functionality of the system enables remote management of household lighting via WiFi, using a web interface.

The web interface empowers users to remotely control lighting, establish schedules, set room-specific lighting activation and deactivation, and access historical data and statistics on lighting usage for specific time frames and rooms.

The user interface consists of 4 tabs: rooms, scheduler, history, statistics.

The "Rooms" page displays a list of rooms shown as blocks. Each block has details about the name of the room and includes a switch button to control the light. Users can use it to turn the light on or off.

The "Scheduler" page displays a list of planned tasks that control when the lights will be turned on or off at specific times and in specific rooms.

At the top-right corner, there is a button to create a new task for the schedule. When setting up a new task, users choose the room, the desired status (on or off), the date, and the time for the automatic lighting control. For tasks already created, users can edit them by changing the date, time, lighting status, and room, or delete the task entirely.

On the "History" page, users can review the record of when the lights were automatically switched on or off according to (based on) the schedule. Users can also filter the history by selecting a date and room to narrow down the results.

The "Statistics" page allows users to view data about how much time the light was either on or off in specific rooms or across all rooms during a defined timeframe. Users can filter the statistics by selecting a particular room, timeframe (year, month, week, or day), and measurement unit (month names, specific days, days of the week, or hours of the day). The information is presented in a graph format for easy visualization.

A web emulator was developed to simulate the functionality of the system.

The central control unit, constituting the control system, is implemented on the basis of a microcontroller. This microcontroller acts as an intermediary, providing communication among lighting fixtures and other system devices.

  • Technologies Used
  • HTML, CSS, JavaScript, React, Node.js, Express.js, REST API, SQL, MySQL, microcontroller ESP32
  • https://github.com/JuliaGood/diploma-j-kuzm
